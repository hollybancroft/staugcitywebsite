Glossary + Works Cited

val()

Get the value of a text box or other form control

text

Get the text of an element.

next([type])

Get the next sibling of an element or the next sibling of a specified type if the parameter is coded.

submit()

Submit the selected form

focus()

Move the focus to the selected form control or link

rev()

Get the previous sibling of each selected element

find()

Search the selected element and return descendant elements

attr()

Get the value of the specified attribute from the first selected element

css()

Get the value of the specified property for each selected element

addClass()

Add one or more classes to the selected elements, and, if necessary, create the class

removeClass()

Remove one or more classes.

toggleClass()

If the class is present, remove it. Otherwise, add it.

html()

Sets the HTML contents of each selected element to the specified HTML string.

hide()

Hide the selected elements

show()

Show the selected elements

each()

Run the function for each element in an array

ready()

The handler runs when the DOM is ready

click()

The handler runs when the selected element is clicked

mouseover()

The handler runs when the mouse pointer moves over the selected element

hover(handlerIn, handlerOut)

The first event handler runs when the mouse pointer moves into an element. The second event handler runs when the mouse pointer moves out.

animate()

Modifies the selected element by changing properties and giving the illusion of animation

delay()

Delay the start of the next animation in the queue

stop()

Stop the current animation for the selected element

finish()

Stop the current animation for the selected element, clear the queue, and compelte all animations for the selected elements.
